Detailed Description

template<typename... HandlerArgs>
class nozomi::StreamingHTTPHandler< HandlerArgs >

Creates a handler for streaming HTTP data. This is more useful for larger requests that may not all fit in memory at once

Template Parameters
HandlerArgsArguments parsed out from the HTTP request per the logic in the Route that created this handler
